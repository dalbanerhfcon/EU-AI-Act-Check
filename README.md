# EU AI Act Quick Check für KMU – Paket mit Zurück-Button

## Inhalt
- vollständige Web-App
- aktuelles app.js mit:
  - vereinfachter Ergebnisansicht
  - Handlungsempfehlungen rechts
  - ohne separaten Betroffenheitsblock
  - ohne „Nächster sinnvoller Schritt“
  - Zurück-Button auf Intro, Fragen und Ergebnis

## Start
1. ZIP entpacken.
2. `index.html` im Browser öffnen.
3. Nach Änderungen im Browser mit Strg + F5 neu laden.

## Kiosk-Modus Edge
```bat
msedge.exe --kiosk "file:///C:/eu-ai-act-check-v3-zurueck-button/index.html" --edge-kiosk-type=fullscreen
```
